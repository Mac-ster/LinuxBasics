Q1
Command to used to display the system’s date and time
Ans
date
Q2
Command to display only the date of the system
Ans
date “+%m-%d-%y” or date “+%D”
Q3
Command to display only the time of the system
Ans
date “+%H:$M:%S”
Q4
Command to display current day
Ans
date “%+A”
Q5
Syntax to print “TODAY IS :(day)”
Ans
date “+Today is %A”
Q6
Command to print the current year
Ans
date “+%Y”
Q7
Command to print current day of the week (numeric)
Ans
date “+%u”
Q8
Command to display date in the standard format
Ans
date -I
Q9
format specifier for minute
Ans
date “+%M”
Q10
format specifier for nanoseconds
Ans
date “+%N”
