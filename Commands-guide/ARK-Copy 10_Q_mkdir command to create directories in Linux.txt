Q1	what is the command to make directory in Linux
Ans	Mkdir
Q2	what does mkdir stand for
Ans	Make Directory
Q3	sample syntax for mkdir command
Ans	Mkdir directory-name
Q4	what does mkdir xyz do
Ans	Create new directory name xyz
Q5	Shortcut to create a subdirectory in home directory
Ans	Mkdir -p home/subdirectory
Q6	how to make a nested subdirectory
Ans	Mkdir -p a/b
Q7	Command to create a directory named photos
Ans	Mkdir photos
Q8	Command to make a directory and subdirectory names/mark
Ans	Mkdir -p names/mark
Q9	Command to make a directory vids in home folder
Ans	Mkdir ~/vids
Q10	Command to make multiple nested subdirectories (e.g.- a1/a2/a3/a4/a5)
Ans	Mkdir -p a1/a2/a3/a4/a5
